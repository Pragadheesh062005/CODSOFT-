# 📱 Mobile App Signup Flow (Figma Design)

This repository contains the **UI/UX design** for a **Mobile App Signup Flow**, created in **Figma**.  
The goal of this design is to demonstrate a clean, simple, and user-friendly signup experience.

---

## 📂 Files in this Repository
- `Mobile App Signup (1).fig` → The original Figma design file.
- `assets/` → Exported design assets (optional: PNG/SVGs if added).
- `README.md` → Project documentation.

---

## 🎨 Design Overview
The signup flow includes:
- **Welcome Screen** – App branding and introduction.
- **User Input Screens** – Forms for name, email, phone, and password.
- **Validation Messages** – Error prompts for incorrect inputs.
- **Success Screen** – Confirmation after successful signup.
